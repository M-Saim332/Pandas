# Pandas Practice 🐼

Practice exercises covering core Pandas concepts for Data Science.

## Topics Covered
- Series & DataFrames — creation, access, update
- Importing Data — CSV and JSON files
- Selection & Indexing — .loc[] and .iloc[]
- Filtering — conditional selection with AND/OR
- Aggregation & GroupBy — mean, sum, min, max, count
- Data Cleaning — NaN handling, dropna(), fillna()
- Data Manipulation — rename, astype(), new columns
- Merging & Concatenation — pd.merge(), pd.concat()

## Libraries Used
- Pandas

## How to Run
Open any `.ipynb` file in Jupyter Notebook or VS Code
and run cells sequentially.
